### Quick Start

1. Navigate to the `site-cookies/` directory
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the `site-cookies` folder
5. Pin the extension and start browsing!
